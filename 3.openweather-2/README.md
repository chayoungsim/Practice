# 날씨앱: OpenWeatherMap API
- 앱이실행되자마자 현재 위치의 날씨 정보가 보인다
- 5개의 버튼 현재위치, 다른도시
- 도시버튼을 클릭할때마다 도시별 날씨가 나온다
- 현재위치 버튼을 누리면 다시 현재 위치 기반의 날씨가 나온다
- 데이터를 가져오는동안 로딩스피너가 돈다

### 설치
 - SCSS : npm install -D sass
 - Axios : npm install axios (API를 처리하는 가장 일반적인 방법)
